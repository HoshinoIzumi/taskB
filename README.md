# Todo Web App
This is a simple todo web app that will help you keep track of your tasks.

## Tech Stack
- Backend: .NET 10 Web API
- Frontend: next.js
- Database: SQL Server

## How to run

### Backend

`cd backend`

`dotnet restore`

`dotnet build`

`dotnet run --project src/TaskB.API`


### Frontend

`cd frontend`

`npm install`

`npm run dev`
